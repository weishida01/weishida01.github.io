1. 研发背景

在FPC生产中，由于对位贴合不精准，造成FPC曝光不精确，造成FPC成品缺陷，影响生产率，故增加CCD对位，精准对位，提高FPC生产良率。

2. 系统解决方案

  1. 系统名称：基于UVW平台的4相机FPC曝光机视觉对位方案
  2. 解决方案：根据FPC曝光机的生产工艺，通过在FPC上方安装4个上相机，实现4上相机的自动在线引导，根据要求，进行对位贴合检测，有效提高产品品质和生产效率。
  3. 检测方式：对FPC进行位置检测，对上下平台曝光网版，进行在线全自动、非接触引导对位，并测量对位精度。
  4. 视觉主要功能：视觉检测FPC位置，将FPC 偏差值传给UVW平台，引导UVW平台进行贴合；视觉测量FPC位置精度

3. 技术指标

针对当前标准配置一拖四的方式，采用 4个 500 万相机，技术指标如下

技术项 技术参数 验收指标
精度要求 视野16.8×14mm，像素当量为 6.8um/pixel 对位精度≤±5um 视野范围>12.6×9.8mm
效率要求 4 个相机同时拍照，1个视觉控制器处理，4个相机综合输出1个结果发送给 UVW平台，实现对位，检测 三次时间不超过3s
检测内容 相机自动寻找靶点，实现对位与精度检测，相机，背光，兼容250-610mm尺寸 兼容250-610mm尺寸检测

检测原理：

  1. 用到4个相机，联合标定，统一到一个图像坐标系中
  2. 已知初始，UVW可以带动上下平台通过UV移动精准贴合到FPC上的，初始FPC位置，且UVW平台旋转中心在上下平台中心
  3. 四个相机拍四个角的靶点，识别出4个靶点的圆心，对角圆心，识别为2条对角线，两条对角线交点为FPC中心
  4. 以对角线其中的一条，与初始模版对角线计算偏差，计算出角度偏差
  5. 以对角线交点，与初始模版的对角线交点做平移偏差，计算出平移偏差
  6. 角度偏差与平移偏差，给到UVW平台，带动上下平台移动

7.2 直线运动

7.2.1 工作台无倾斜时(θ0=0时)
7.2.1.1 向水平轴和垂直轴方向平行移动

工作台无倾斜时，各轴模块的移动量即为工作台的移动量。

7.2.1.2 X-Y 平行移动

工作台无倾斜时，各轴模块移动量的总和即为工作台的移动量。

工作台的移动量为X轴方向的移动量和Y轴的移动量的综合

7.2.2 工作台有倾斜时(θ0≠0时)
7.2.2.1 向倾斜方向平行移动

当工作台有倾斜（θ0）时，如果使各轴的模块向 X 轴方向或 Y 轴方向移动，工作台则向该倾斜方向移动。

可用工作台倾斜方向的移动量 S 除以 Cosθ0，求出各轴模块的移动量

7.2.2.2 向水平轴和垂直轴方向平行移动

为了在保持工作台倾斜的状态下使工作台向水平轴方向或垂直轴方向移动，除了驱动工作台将要移动的方向的模块，还需要驱动与其相对的垂直方向的模块，以抵消工作台向倾斜方向的移動量

7.3 旋转运动

7.3.1、运动图解，参数定义
  1. UVW校准工作台不仅能以工作台中心为旋转中心旋转，还能以工作台上的任意位置为旋转中心旋转。
  2. 与旋转相关的计算公式非常复杂，但基本上可通过从工作台旋转后的位置中减去旋转前的模块的位置来求出各轴模块的移动量
  3. 关于各轴具体的移动量，需要以工作台的旋转量为基础，根据后述计算公式求出。

7.3.2、以工作台中心为旋转中心作旋转运动

  1. 工作台当前位置（0,0,0），工作台移动量（Sx，Sy，Sθ）
  2. 旋转中心一直为工作台中心时，Rx1，Rx2，Ry1，Ry2，θX1，θX2，θY1，θY2，8个参数固定不变

  1. 工作台当前位置（x0,y0,θ0），工作台移动量（Sx，Sy，Sθ）

#      旋转中心一直为平台中心,工作台当前位置（x0,y0,θ0）,工作台移动（Sx，Sy，Sθ）
import math

# 定义8轴平台基准参数类 与移动参数
# 输入平台8轴位置参数
x1_mark_x,x2_mark_x,x3_mark_x,x4_mark_x = 0,    0,      0,  67.5
x1_mark_y,x2_mark_y,x3_mark_y,x4_mark_y = 0,    0,      0,  -59.0
y1_mark_x,y2_mark_x,y3_mark_x,y4_mark_x = 59.0, -59.0,  0,  0
y1_mark_y,y2_mark_y,y3_mark_y,y4_mark_y = 67.5, 67.5,   0,  0
# 输入平台初始位置，与偏移量
x0,     y0,     angle0  =   0,     0,     0 * math.pi / 180       #初始位置
s_x,    s_y,    s_angle =   0,     0,     2 * math.pi / 180       #偏移量

#8轴的偏移计算函数
def x1_offset(s_x,s_y,s_angle,x0,y0,angle0):
    s_x = s_x + x0
    s_y = s_y + y0
    s_angle = s_angle + angle0
    rx_1 = math.sqrt(x1_mark_x * x1_mark_x + x1_mark_y * x1_mark_y)
    angle_x1 = math.atan2(abs(x1_mark_y), abs(x1_mark_x))
    x1_offset_1 = -rx_1 * math.cos(angle_x1) +rx_1 * math.cos(s_angle + angle_x1) + \
                (rx_1 * math.sin(s_angle + angle_x1) - rx_1 * math.sin(angle_x1)) * math.tan(s_angle) + \
                s_x + s_y * math.tan(s_angle)
    x1_offset1_2 = -rx_1 * math.cos(angle_x1) +rx_1 * math.cos(angle0 + angle_x1) + \
                (rx_1 * math.sin(angle0 + angle_x1) - rx_1 * math.sin(angle_x1)) * math.tan(angle0) + \
                x0 + y0 * math.tan(angle0)
    return x1_offset_1 - x1_offset1_2
def x2_offset(s_x,s_y,s_angle,x0,y0,angle0):
    s_x = s_x + x0
    s_y = s_y + y0
    s_angle = s_angle + angle0
    rx_2 = math.sqrt(x2_mark_x * x2_mark_x + x2_mark_y * x2_mark_y)
    angle_x2 = math.atan2(abs(x2_mark_y), abs(x2_mark_x))
    x2_offset_1 = rx_2 * math.cos(angle_x2) -rx_2 * math.cos(-s_angle + angle_x2) - \
                (-rx_2 * math.sin(-s_angle + angle_x2) + rx_2 * math.sin(angle_x2)) * math.tan(s_angle) + \
                s_x + s_y * math.tan(s_angle)
    x2_offset_2 = rx_2 * math.cos(angle_x2) - rx_2 * math.cos(-angle0 + angle_x2) - \
                (-rx_2 * math.sin(-angle0 + angle_x2) + rx_2 * math.sin(angle_x2)) * math.tan(angle0) + \
                x0 + y0 * math.tan(angle0)
    return x2_offset_1-x2_offset_2
def x3_offset(s_x,s_y,s_angle,x0,y0,angle0):
    s_x = s_x + x0
    s_y = s_y + y0
    s_angle = s_angle + angle0
    rx_3 = math.sqrt(x3_mark_x * x3_mark_x + x3_mark_y * x3_mark_y)
    angle_x3 = math.atan2(abs(x3_mark_y), abs(x3_mark_x))
    x3_offset_1 = rx_3 * math.cos(angle_x3) -rx_3 * math.cos(s_angle + angle_x3) - \
                (rx_3 * math.sin(s_angle + angle_x3) - rx_3 * math.sin(angle_x3)) * math.tan(s_angle) + \
                s_x + s_y * math.tan(s_angle)
    x3_offset_2 = rx_3 * math.cos(angle_x3) - rx_3 * math.cos(angle0 + angle_x3) - \
                  (rx_3 * math.sin(angle0 + angle_x3) - rx_3 * math.sin(angle_x3)) * math.tan(angle0) + \
                  x0 + y0 * math.tan(angle0)
    return x3_offset_1 - x3_offset_2
def x4_offset(s_x,s_y,s_angle,x0,y0,angle0):
    s_x = s_x + x0
    s_y = s_y + y0
    s_angle = s_angle + angle0
    rx_4 = math.sqrt(x4_mark_x * x4_mark_x + x4_mark_y * x4_mark_y)
    angle_x4 = math.atan2(abs(x4_mark_y), abs(x4_mark_x))
    x4_offset_1 = - rx_4 * math.cos(angle_x4) + rx_4 * math.cos(-s_angle + angle_x4) + \
                (-rx_4 * math.sin(-s_angle + angle_x4) + rx_4 * math.sin(angle_x4)) * math.tan(s_angle) + \
                s_x + s_y * math.tan(s_angle)
    x4_offset_2 = - rx_4 * math.cos(angle_x4) + rx_4 * math.cos(-angle0 + angle_x4) + \
                  (-rx_4 * math.sin(-angle0 + angle_x4) + rx_4 * math.sin(angle_x4)) * math.tan(angle0) + \
                  x0 + y0 * math.tan(angle0)
    return x4_offset_1-x4_offset_2
def y1_offset(s_x,s_y,s_angle,x0,y0,angle0):

    s_y = s_y + y0
    s_angle = s_angle + angle0
    ry_1 = math.sqrt(y1_mark_x * y1_mark_x + y1_mark_y * y1_mark_y)

    y1_offset_1 = -ry_1 * math.sin(angle_y1) + ry_1 * math.sin(s_angle + angle_y1) + \
                (-ry_1 * math.cos(s_angle + angle_y1) + ry_1 * math.cos(angle_y1)) * math.tan(s_angle) + \
                s_y - s_x * math.tan(s_angle)
    y1_offset_2 = -ry_1 * math.sin(angle_y1) + ry_1 * math.sin(angle0 + angle_y1) + \
                  (-ry_1 * math.cos(angle0 + angle_y1) + ry_1 * math.cos(angle_y1)) * math.tan(angle0) + \
                  y0 - x0 * math.tan(angle0)
    return y1_offset_1 - y1_offset_2

    y2_offset_2 = -ry_2 * math.sin(angle_y2) + ry_2 * math.sin(-angle0 + angle_y2) + \

                  (-ry_3 * math.cos(angle0 + angle_y3) + ry_3 * math.cos(angle_y3)) * math.tan(angle0) + \
                  y0 - x0 * math.tan(angle0)
    return y3_offset_1 - y3_offset_2
def y4_offset(s_x,s_y,s_angle,x0,y0,angle0):
    s_x = s_x + x0
    s_y = s_y + y0
    s_angle = s_angle + angle0
    ry_4 = math.sqrt(y4_mark_x * y4_mark_x + y4_mark_y * y4_mark_y)
    angle_y4 = math.atan2(abs(y4_mark_y), abs(y4_mark_x))
    y4_offset_1 = ry_4 * math.sin(angle_y4) - ry_4 * math.sin(-s_angle + angle_y4) - \
                (ry_4 * math.cos(-s_angle + angle_y4) - ry_4 * math.cos(angle_y4)) * math.tan(s_angle) + \
                s_y - s_x * math.tan(s_angle)
    y4_offset_2 = ry_4 * math.sin(angle_y4) - ry_4 * math.sin(-angle0 + angle_y4) - \
                  (ry_4 * math.cos(-angle0 + angle_y4) - ry_4 * math.cos(angle_y4)) * math.tan(angle0) + \
                  y0 - x0 * math.tan(angle0)
    return y4_offset_1 - y4_offset_2

def result():
    print('工作台移动前位置', end='\t\t')

    print('工作台偏移量', end='\t\t\t')
    print('x=' + str(s_x) + ' y=' + str(s_y) + ' θ=' + str(180 * (s_angle) / math.pi))

    print('x='+str(s_x+x0)+' y='+str(s_y+y0)+' θ='+str(180*(s_angle+angle0)/math.pi))

    print('u='+str(x4_offset(s_x,s_y,s_angle,x0,y0,angle0)), end='\t\t')

    print('w='+str(y1_offset(s_x,s_y,s_angle,x0,y0,angle0)))
