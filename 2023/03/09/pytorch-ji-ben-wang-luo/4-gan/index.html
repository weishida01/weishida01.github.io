1. GAN 介绍

生成式对抗网络

GAN的基本原理其实非常简单，这里以生成图片为例进行说明。假设我们有两个网络，G（Generator）和D（Discriminator）。正如它的名字所暗示的那样，它们的功能分别是：

  • G是一个生成图片的网络，它接收一个随机的噪声z，通过这个噪声生成图片，记做G(z)。
  • D是一个判别网络，判别一张图片是不是“真实的”。它的输入参数是x，x代表一张图片，输出D（x）代表x为真实图片的概率，如果为1，就代表100%是真实的图片，而输出为0，就代表不可能是真实的图片。

在训练过程中，生成网络G的目标就是尽量生成真实的图片去欺骗判别网络D。而D的目标就是尽量把G生成的图片和真实的图片分别开来。这样，G和D构成了一个动态的“博弈过程”。

最后博弈的结果是什么？在最理想的状态下，G可以生成足以“以假乱真”的图片G(z)。对于D来说，它难以判定G生成的图片究竟是不是真实的，因此D(G(z)) = 0.5。

公式

简单分析一下这个公式：

  • 整个式子由两项构成。x表示真实图片，z表示输入G网络的噪声，而G(z)表示G网络生成的图片。
  • D(x)表示D网络判断真实图片是否真实的概率（因为x就是真实的，所以对于D来说，这个值越接近1越好）。而D(G(z))是D网络判断G生成的图片的是否真实的概率。
  • G的目的：上面提到过，D(G(z))是D网络判断G生成的图片是否真实的概率，G应该希望自己生成的图片“越接近真实越好”。也就是说，G希望D(G(z))尽可能得大，这时V(D, G)会变小。因此我们看到式子的最前面的记号是min_G。
  • D的目的：D的能力越强，D(x)应该越大，D(G(x))应该越小。这时V(D,G)会变大。因此式子对于D来说是求最大(max_D)

·难以训练

3. CGAN

CGAN（条件生成对抗网络）

针对问题：模型过于自由不可控。

方法：输入更多信息到GAN模型学习，生成更好的样本。

效果：提高生成图像的质量，明确控制图像的某些方面。

4. DCGAN原理介绍

结合了有监督学习的CNN和无监督的GAN

针对问题：GAN训练不稳定，经常生成无意义的输出。

方法：生成模型和判别模型均采用CNN模型，并在结构上做了一些改变。

我们知道深度学习中对图像处理应用最好的模型是CNN，那么如何把CNN与GAN结合？DCGAN是这方面最好的尝试之一（论文地址：[1511.06434] Unsupervised Representation Learning with Deep Convolutional Generative Adversarial Networks）

DCGAN的原理和GAN是一样的，这里就不在赘述。它只是把上述的G和D换成了两个卷积神经网络（CNN）。但不是直接换就可以了，DCGAN对卷积神经网络的结构做了一些改变，以提高样本的质量和收敛的速度，这些改变有：

  • 取消所有pooling层。G网络中使用转置卷积（transposed convolutional layer）进行上采样，D网络中用加入stride的卷积代替pooling。
  • 在D和G中均使用batch normalization
  • 去掉FC层，使网络变为全卷积网络
  • G网络中使用ReLU作为激活函数，最后一层使用tanh
  • D网络中使用LeakyReLU作为激活函数

5.1 原始GAN训练困难的分析

1）训练目标：

x表示真实图片，z表示输入G网络的噪声，而G(z)表示G网络生成的图片。

判别器LOSS（最小化）：

生成器LOSS（最小化）：

2）训练过程：

· 先固定生成器，训练判别器达到最优，然后训练生成器。

· 利用SGD训练判别器达到最优解为：

训练生成器（在判别器最优时）：最终变换形式：即最小化Pr和Pg之间的JS散度。由于P_r与P_g几乎不可能有不可忽略的重叠，所以无论它们相距多远JS散度都是常数log 2，最终导致生成器的梯度（近似）为0，梯度消失。

最小化目标分析：u最小化生成分布与真实分布的KL散度，却又要最大化两者的JS散度，在数值上则会导致梯度不稳定。uKL散度会造成两种错误：生成器没能生成真实样本（缺乏多样性），生成器生成不真实样本（缺乏准确性）

小结：在原始GAN的（近似）最优判别器下，第一种生成器loss面临梯度消失问题，第二种生成器loss面临优化目标荒谬、梯度不稳定、对多样性与准确性惩罚不平衡导致mode collapse这几个问题。

5.2 WGAN的贡献

Wasserstein距离：（EM距离）相比KL散度、JS散度的优越性：即便两个分布没有重叠，Wasserstein距离仍能反映它们的远近。KL散度和JS散度是突变的，Wasserstein距离却是平滑的，可以提供有意义的梯度。

EM距离相对KL散度与JS散度具有优越的平滑特性，理论上可以解决梯度消失问题。在此近似最优判别器下优化生成器使得Wasserstein距离缩小，就能有效拉近生成分布与真实分布。WGAN既解决了训练不稳定的问题，也提供了一个可靠的训练进程指标，而且该指标确实与生成样本的质量高度相关。

6. 原生GAN实战例子

生成鉴别高斯混合模型数据，此处不是图片数据

真实x为高斯点数据

假数据经过G生成X

原生GAN问题：由于 x真实数据与 X假数据不重叠，训练不起来，D loss一直为0，G loss一直为-1

import  torch
from    torch import nn, optim, autograd
import  numpy as np
import  visdom
import  random

torch.manual_seed(23)
np.random.seed(23)


# 1、定义参数
h_dim = 400
batchsz = 512
viz = visdom.Visdom()   # 可视化


# 2、生成数据集
# 已知的8个gaussian mixture models
def data_generator():

    scale = 2.
    centers = [                             # 8个gaussian mixture models
        (1, 0),
        (-1, 0),
        (0, 1),
        (0, -1),
        (1. / np.sqrt(2), 1. / np.sqrt(2)),
        (1. / np.sqrt(2), -1. / np.sqrt(2)),
        (-1. / np.sqrt(2), 1. / np.sqrt(2)),
        (-1. / np.sqrt(2), -1. / np.sqrt(2))
    ]
    centers = [(scale * x, scale * y) for x, y in centers]
    while True:
        dataset = []
        for i in range(batchsz):                # 生成一个batch的数据集
            point = np.random.randn(2) * .02     # 生成一个点
            center = random.choice(centers)
            point[0] += center[0]               # 点符合为8个分布中的一个
            point[1] += center[1]
            dataset.append(point)               # 点加入数据集
        dataset = np.array(dataset, dtype='float32')   # 转换类型
        dataset /= 1.414  # stdev
        yield dataset


# 3、生成器模型
class Generator(nn.Module):

    def __init__(self):
        super(Generator, self).__init__()
        # z:[b,2] => [b,2]   gan的z输入 输出为x 此处不为图片 为（x1,x2）坐标
        self.net = nn.Sequential(
            nn.Linear(2, h_dim),
            nn.ReLU(True),
            nn.Linear(h_dim, h_dim),
            nn.ReLU(True),
            nn.Linear(h_dim, h_dim),
            nn.ReLU(True),
            nn.Linear(h_dim, 2),
        )

    def forward(self, z):
        output = self.net(z)
        return output


# 4、鉴别器模型
class Discriminator(nn.Module):

    def __init__(self):
        super(Discriminator, self).__init__()
        self.net = nn.Sequential(
            nn.Linear(2, h_dim),
            nn.ReLU(True),
            nn.Linear(h_dim, h_dim),
            nn.ReLU(True),
            nn.Linear(h_dim, h_dim),
            nn.ReLU(True),
            nn.Linear(h_dim, 1),
            nn.Sigmoid()
        )

    def forward(self, x):
        output = self.net(x)
        return output.view(-1)


# 5、主函数 训练模型
def main():

    # 测试生成的真实数据
    data_iter = data_generator()
    print('batch:', next(data_iter).shape)

    # 实例化网络结构
    G = Generator().cuda()
    D = Discriminator().cuda()

    # 定义优化器
    optim_G = optim.Adam(G.parameters(), lr=1e-3, betas=(0.5, 0.9))  # 经验参数 betas=(0.5, 0.9)
    optim_D = optim.Adam(D.parameters(), lr=1e-3, betas=(0.5, 0.9))

    # 可视化 划线
    viz.line([[0,0]], [0], win='loss', opts=dict(title='loss',legend=['D', 'G']))  # 可视化 2个曲线

    for epoch in range(50000):
        # 1. train discriminator for k steps  训练discriminator K次
        for _ in range(5):
            x = next(data_iter)                 # 训练真实数据
            xr = torch.from_numpy(x).cuda()

            # [b，2] =》 [b,1]   二维的点数据，输出鉴别的结果
            predr = (D(xr))
            lossr = - (predr.mean())   # 最大化 max log(lossr)  由于随机梯度下降 加-号

            # [b, 2]
            z = torch.randn(batchsz, 2).cuda()      # 生成的假数据
            xf = G(z).detach()    # 由于之训练discriminator 但把Generator也加进来了，所以要.detach() 类似stop gradient 不计算梯度
            predf = (D(xf))
            lossf = (predf.mean())      # 最小化 min predf
            loss_D = lossr + lossf

            optim_D.zero_grad()
            loss_D.backward()
            optim_D.step()

        # 2. train Generator               # 训练Generator 1次
        z = torch.randn(batchsz, 2).cuda()   # 生成假数据
        xf = G(z)
        predf = D(xf)                       # 由于是向后传，此处不能加.detach()

        loss_G = - (predf.mean())           # max predf
        optim_G.zero_grad()
        loss_G.backward()
        optim_G.step()


        if epoch % 100 == 0:
            # 可视化曲线
            viz.line([[loss_D.item(), loss_G.item()]], [epoch], win='loss', update='append')

            print(loss_D.item(), loss_G.item())


if __name__ == '__main__':
    main()

7.3 代码

'''
WGAN
生成鉴别高斯混合模型数据，此处不是图片数据
真实x为高斯点数据
假数据经过G生成X‘

原生GAN问题
1、由于 x真实数据与 X假数据不重叠，训练不起来，D loss一直为0，G loss一直为-1

解决用WGAN
1、加入惩罚项 gradient_penalty   loss_D = lossr + lossf + gp
2、初始化参数 D G 参数  可不加

'''


import  torch
from    torch import nn, optim, autograd
import  numpy as np
import  visdom
import  random

torch.manual_seed(23)
np.random.seed(23)


# 1、定义参数
h_dim = 400
batchsz = 512
viz = visdom.Visdom()   # 可视化


# 2、生成数据集
# 已知的8个gaussian mixture models
def data_generator():

    scale = 2.
    centers = [                             # 8个gaussian mixture models
        (1, 0),
        (-1, 0),
        (0, 1),
        (0, -1),
        (1. / np.sqrt(2), 1. / np.sqrt(2)),
        (1. / np.sqrt(2), -1. / np.sqrt(2)),
        (-1. / np.sqrt(2), 1. / np.sqrt(2)),
        (-1. / np.sqrt(2), -1. / np.sqrt(2))
    ]
    centers = [(scale * x, scale * y) for x, y in centers]
    while True:
        dataset = []
        for i in range(batchsz):                # 生成一个batch的数据集
            point = np.random.randn(2) * .02     # 生成一个点
            center = random.choice(centers)
            point[0] += center[0]               # 点符合为8个分布中的一个
            point[1] += center[1]
            dataset.append(point)               # 点加入数据集
        dataset = np.array(dataset, dtype='float32')   # 转换类型
        dataset /= 1.414  # stdev
        yield dataset


# 3、生成器模型
class Generator(nn.Module):

    def __init__(self):
        super(Generator, self).__init__()
        # z:[b,2] => [b,2]   gan的z输入 输出为x 此处不为图片 为（x1,x2）坐标
        self.net = nn.Sequential(
            nn.Linear(2, h_dim),
            nn.ReLU(True),
            nn.Linear(h_dim, h_dim),
            nn.ReLU(True),
            nn.Linear(h_dim, h_dim),
            nn.ReLU(True),
            nn.Linear(h_dim, 2),
        )

    def forward(self, z):
        output = self.net(z)
        return output


# 4、鉴别器模型
class Discriminator(nn.Module):

    def __init__(self):
        super(Discriminator, self).__init__()
        self.net = nn.Sequential(
            nn.Linear(2, h_dim),
            nn.ReLU(True),
            nn.Linear(h_dim, h_dim),
            nn.ReLU(True),
            nn.Linear(h_dim, h_dim),
            nn.ReLU(True),
            nn.Linear(h_dim, 1),
            nn.Sigmoid()
        )

    def forward(self, x):
        output = self.net(x)
        return output.view(-1)


# 5、WGAN的 GP惩罚项设置
def gradient_penalty(D, xr, xf):
    """
    :param D:
    :param xr: [b,2]
    :param xf: [b,2]
    :return: gp
    """
    LAMBDA = 0.3

    # only constrait for Discriminator
    xf = xf.detach()      # 不需要对G求导
    xr = xr.detach()

    # [b, 1] => [b, 2]
    alpha = torch.rand(batchsz, 1).cuda()   # 生成数据
    alpha = alpha.expand_as(xr)             # 扩展成类似 xr的维度
    interpolates = alpha * xr + ((1 - alpha) * xf)

    interpolates.requires_grad_()          # 设置为有导数信息
    disc_interpolates = D(interpolates)
    # create_graph 二阶求导设置 retain_graph 图需要backword需要
    gradients = autograd.grad(outputs=disc_interpolates, inputs=interpolates,
                              grad_outputs=torch.ones_like(disc_interpolates),
                              create_graph=True, retain_graph=True, only_inputs=True)[0]
    # 总的线性插值求和
    gp = ((gradients.norm(2, dim=1) - 1) ** 2).mean() * LAMBDA    # 求2范数，-1 平方 求均值

    return gp


# 6、初始化参数 D G 参数
def weights_init(m):
    if isinstance(m, nn.Linear):
        # m.weight.data.normal_(0.0, 0.02)
        nn.init.kaiming_normal_(m.weight)
        m.bias.data.fill_(0)


# 7、训练主函数
def main():

    # 1、测试生成的真实数据
    data_iter = data_generator()
    print('batch:', next(data_iter).shape)

    # 2、实例化网络结构
    G = Generator().cuda()
    D = Discriminator().cuda()
    G.apply(weights_init)
    D.apply(weights_init)

    # 3、定义优化器
    optim_G = optim.Adam(G.parameters(), lr=1e-3, betas=(0.5, 0.9))  # 经验参数 betas=(0.5, 0.9)
    optim_D = optim.Adam(D.parameters(), lr=1e-3, betas=(0.5, 0.9))

    # 4、可视化 loss 结果
    viz.line([[0,0]], [0], win='loss', opts=dict(title='loss',legend=['D', 'G']))  # 可视化 2个曲线

    # 5、训练 epoch
    for epoch in range(50000):

        # 1. 训练discriminator K次
        for _ in range(5):
            x = next(data_iter)                 # 训练真实数据
            xr = torch.from_numpy(x).cuda()

            # 1、输出鉴别结果 [b，2] =》 [b,1]   二维的点数据，输出鉴别的结果
            predr = (D(xr))
            lossr = - (predr.mean())   # 最大化 max log(lossr)  由于随机梯度下降 加-号

            # 2、G的结果 [b, 2]
            z = torch.randn(batchsz, 2).cuda()      # 生成的假数据
            xf = G(z).detach()    # 由于之训练discriminator 但把Generator也加进来了，所以要.detach() 类似stop gradient 不计算梯度
            predf = (D(xf))

            # 3、损失函数设置
            lossf = (predf.mean())      # 最小化 min predf
            # gradient penalty      加入惩罚项
            gp = gradient_penalty(D, xr, xf)
            loss_D = lossr + lossf + gp

            # 4、优化设置
            optim_D.zero_grad()
            loss_D.backward()
            # for p in D.parameters():
            #     print(p.grad.norm())
            optim_D.step()


        # 2. 训练Generator 1次
        z = torch.randn(batchsz, 2).cuda()   # 生成假数据
        xf = G(z)
        predf = D(xf)                       # 由于是向后传，此处不能加.detach()

        loss_G = - (predf.mean())           # max predf
        optim_G.zero_grad()
        loss_G.backward()
        optim_G.step()

        # 3. 每100次循环 可视化，打印结果一次
        if epoch % 100 == 0:
            # 可视化曲线
            viz.line([[loss_D.item(), loss_G.item()]], [epoch], win='loss', update='append')

            print(loss_D.item(), loss_G.item())


if __name__ == '__main__':
    main()
